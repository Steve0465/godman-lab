# godman-lab
Personal automation lab for scripts, agents, and AI projects. Experiments, tools, and workflow systems.

## Tools

### Receipt Processing
OCR, parse, categorize, and summarize scanned receipts.
See [docs/README_receipts.md](docs/README_receipts.md) for details.

### GitHub Code Search
Search for code and repositories on GitHub from the command line.
See [docs/README_github_search.md](docs/README_github_search.md) for details.

Quick start:
```bash
python github_search.py --code "your search query" --language python
python github_search.py --repos "repository search" --sort stars
```
